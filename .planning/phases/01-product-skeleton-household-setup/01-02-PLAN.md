<threats>
- threat_id: T-01-04
  category: Spoofing
  component: `/auth/login` 与 `/auth/me`
  disposition: mitigate
  mitigation_plan: 测试账号密码必须以哈希形式存储，`/auth/login` 只接受手机号与密码组合，`/auth/me` 只从 session 读取 account_id，不接受客户端传入账户标识。
- threat_id: T-01-05
  category: Elevation of Privilege
  component: household / item 受保护接口
  disposition: mitigate
  mitigation_plan: `GET /household/me`、`PATCH /household/{id}`、`GET /items/{id}`、`PATCH /items/{id}`、`/disable`、`/enable` 全部通过 `owner_account_id` 或 `household_id` 回查当前登录账户，禁止跨账户访问。
- threat_id: T-01-06
  category: Information Disclosure
  component: API 响应体
  disposition: mitigate
  mitigation_plan: 登录、会话、家庭和物品响应只返回当前阶段需要的公开字段，不把 `password_hash`、原始 session 内容和其他账户数据写入响应。
</threats>
</threat_model>
