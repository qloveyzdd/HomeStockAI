<threats>
- threat_id: T-01-07
  category: Spoofing
  component: Web 侧登录与页面守卫
  disposition: mitigate
  mitigation_plan: 所有受保护页面都通过统一的 `server-session` 或等价入口调用 `/auth/me` 判断登录态，不允许只靠前端本地状态放行。
- threat_id: T-01-08
  category: Tampering
  component: Household / Item 表单提交
  disposition: mitigate
  mitigation_plan: Web 表单字段必须与 API schema 对齐，枚举字段只允许来自固定选项，`credentials: include` 固定开启，避免手工拼接账户标识或 household_id。
- threat_id: T-01-09
  category: Information Disclosure
  component: 页面导航与设置页
  disposition: mitigate
  mitigation_plan: 设置页只展示当前账号手机号和家庭资料，不渲染密码、哈希值、其他账号或未来阶段的价格/建议信息。
</threats>
</threat_model>
