## 首个价格来源角色

| Option | Description | Selected |
|--------|-------------|----------|
| `什么值得买` 做主来源 | 先验证“值不值得买”的信号能力 | |
| 真实售卖平台做主来源 | 先做真实价格快照主链路 | ✓ |
| 两套来源同时做 | 同时接真实平台和内容信号源 | |

**用户的选择:** 先用真实售卖平台做主链路。
**补充说明:** 首个平台优先 `京东`。`什么值得买` 暂不进入主链路，后续可做增强信号。

## 商品绑定策略

| Option | Description | Selected |
|--------|-------------|----------|
| 用户手动贴京东商品链接 | 明确但输入成本高 | |
| 系统自动搜索匹配 | 减少输入成本 | ✓ |
| 两种都做 | 覆盖更广但复杂度更高 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 1 个主商品 | 最简单 | |
| 2 到 3 个候选商品 | 让用户确认候选池 | ✓ |
| 不限制数量 | 覆盖最多但最重 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 未绑定就不抓价并明确提示 | 范围清晰 | ✓ |
| 自动猜一个商品 | 快但误判风险高 | |
| 低置信度继续抓价 | 结果容易失信 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 系统给 2 到 3 个候选，用户手动确认 | 可控且可信 | ✓ |
| 系统自动绑定最优候选 | 快但容易错绑 | |
| 只给 1 个候选 | 简单但选择空间小 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 只展示高信用/头部品牌候选 | 更稳、更贴近产品气质 | ✓ |
| 所有候选展示但高信用优先 | 覆盖更广 | |
| 纯搜索匹配度排序 | 最省事但信任感弱 | |

**用户的选择:** 自动搜索匹配，支持 `2 到 3` 个候选，用户手动确认；候选池只展示高信用、行业头部、品牌感稳定的商品。
**补充说明:** 用户明确指出客户更像高认知、重品牌信任的人群，不追求“全网最低价”，而是“保留好品牌的同时获得更优惠价格”，心智类似“山姆”。

## 抓价节奏

| Option | Description | Selected |
|--------|-------------|----------|
| 绑定成功后立即抓一次 | 用户马上看到结果 | ✓ |
| 等下一次定时任务 | 更简单但反馈慢 | |
| 用户手动点立即抓价 | 控制强但一步更多 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 每天 2 次 | 更保守 | |
| 每 6 小时一次 | 平衡新鲜度与成本 | ✓ |
| 每小时一次 | 更新更快但成本更高 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 静默失败等下次 | 最省事 | |
| 记录失败并自动重试 1 到 2 次 | 推荐做法 | ✓ |
| 立即前台报错 | 太打扰 | |

**用户的选择:** 绑定后立即抓一次，后续每 6 小时刷新，失败时自动重试 `1 到 2` 次。

## 价格展示入口

| Option | Description | Selected |
|--------|-------------|----------|
| 先放在物品详情页 | 继续沿用现有主工作区 | ✓ |
| 先做独立价格页 | 需要新页面边界 | |
| 两边一起做 | 范围更大 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 不新增一级导航 `优惠` | 保持 `物品 / 设置` | ✓ |
| 新增占位导航 | 提前铺壳 | |
| 直接做完整导航入口 | 越界风险高 | |

| Option | Description | Selected |
|--------|-------------|----------|
| 当前价 + 7 天最低 + 30 天均价 + 是否接近低价 + 最近抓价时间 | 最小可用展示 | ✓ |
| 再加候选商品对比卡片 | 更丰富但更重 | |
| 再加趋势图 | 展示更强但超当前目标 | |

**用户的选择:** 价格先展示在物品详情页，不新增一级导航 `优惠`，首版展示深度只保留最小价格比较信息。
